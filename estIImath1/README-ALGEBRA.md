# Algebra & Functions - EST II Math Level 1

## Unit Overview

This training platform covers the **Algebra & Functions** domain of the EST II High School Math Level 1 exam (2026 specifications). The platform contains **141 practice questions** with detailed solutions and interactive Desmos graphs.

---

## EST Content Standards

### Topics Covered

| Topic | Description |
|-------|-------------|
| **Expressions** | Simplifying, evaluating algebraic expressions |
| **Linear Equations** | Solving single-variable linear equations |
| **Quadratic Equations** | Factoring, quadratic formula, discriminant |
| **Systems of Two Equations** | Substitution, elimination, infinite/no solutions |
| **Simple Inequalities** | Solving linear inequalities |
| **Compound Inequalities** | AND/OR inequalities |
| **Graphing Inequalities (1 variable)** | Number line representation |
| **Graphing Inequalities (2 variables)** | Shading regions, boundary lines |
| **System of Linear Inequalities** | Intersection of shaded regions |
| **Fractions** | Operations, simplification, rational expressions |
| **Percentages** | Percent increase/decrease, calculations |
| **Exponents** | Laws, negative/fractional exponents, radicals |
| **Absolute Values** | Equations and inequalities with |x| |
| **Composition of Functions** | f(g(x)), nested function evaluation |
| **Inverse Functions** | Finding f⁻¹(x), domain/range swap |
| **Polynomial Functions** | Roots, degree, end behavior, graphing |
| **Rational Functions** | Asymptotes, holes, domain, graphing |
